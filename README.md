##### アクセス修飾子  

public - どこからでも使える  
private - そのクラスしか使えない  
protected - 子クラスからはつかえる。継承する時つかう  
virtual - オーバーライドする元の関数につける  
abstract - 実体はない、インターフェイス的にベースクラスに定義するものっぽい
